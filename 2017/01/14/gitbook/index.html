部署与发布

基于 gitbook（发行书籍的社区） 和 github（代码托管，方便fork）。

  • 在github上创建文档仓库

  • 对gitbook进行github的授权

    • Account setting 中 link Github
    • Ps: 如果授权在新建仓库之前，需要点击一下Reconnect Github Account(with access to public reponsitory)

  • 在gitbook上create a new book

    • 选择: BOOK & MANUAL

  • 在新建的book上点击setting

    • 选择Github仓库
    • 增加Webhook（可自动、可手动）
    • 完成后会显示 IN SYNC

  • 之后在Github上推送代码，即可实现自动发布
